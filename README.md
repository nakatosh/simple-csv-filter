# CSV抽出アプリ (CSV Filter App)

検索ワードを入力してCSVファイルをドラッグ＆ドロップするだけで、該当する行のみを抽出した新しいCSVファイルをダウンロードできるブラウザツールです。
PWA（Progressive Web App）に対応しており、スマホやPCにインストールしてオフラインで利用することも可能です。

## ✨ 特徴 (Features)

* **簡単操作**: キーワードを入力して、CSVをドラッグ＆ドロップするだけ。
* **文字コード自動判別**: Excelで作成された「Shift_JIS」と、Web標準の「UTF-8」を自動で判別して読み込むため、文字化けの心配がありません。
* **出力フォーマット**: 出力されるCSVは、どの環境でも文字化けしにくい「BOM付きUTF-8」で統一されます。
* **オフライン動作 (PWA)**: 一度読み込めば、インターネットに繋がっていない環境でも利用可能です。
* **インストール可能**: ブラウザの機能を使って、デスクトップアプリやスマホアプリのようにホーム画面に追加できます。

## 📁 ファイル構成

本アプリは以下のファイルで構成されています。

* `index.html` : アプリのメイン画面と抽出処理のプログラム
* `manifest.json` : PWAの基本情報（アプリ名、アイコンなど）を定義する設定ファイル
* `sw.js` : オフライン動作を実現するためのService Workerプログラム
* `icon-192.png` : アプリアイコン（192x192ピクセル）
* `icon-512.png` : アプリアイコン（512x512ピクセル）

## 🚀 使い方 (Usage)

1. 本アプリ（ `index.html` ）をブラウザで開きます。
2. 画面上の入力欄に、抽出したい**検索ワード**を入力します。
3. 読み込ませたい**CSVファイル**を、画面の点線枠内（「ここにCSVをドラッグ＆ドロップ」）にドラッグ＆ドロップします。
4. 検索ワードが含まれる行のみが抽出され、`filtered_元のファイル名.csv` という名前で自動的にダウンロードされます。
   ※CSVの1行目（ヘッダー行）は常に残る仕様です。

## 📱 PWAとしてインストールするには

PWAの機能（インストール機能やオフライン動作）を有効にするには、ファイルを単にダブルクリックで開く（`file:///~`）のではなく、**Webサーバー経由（`http://~` または `https://~`）でアクセス**する必要があります。

### 開発・テスト環境の場合
VS Codeの拡張機能「Live Server」や、Node.jsの「http-server」などを使用して、ローカルサーバーを立ち上げてアクセスしてください。

### 本番環境の場合
GitHub Pages、Vercel、Netlifyなどの無料ホスティングサービスや、ご自身のWebサーバーにすべてのファイルをアップロードしてください（HTTPS接続必須）。

アクセス後、ブラウザのアドレスバー右端（またはブラウザのメニュー）に表示される「インストール」ボタンを押すことで、アプリとして端末に追加できます。
